# Instrucciones de despliegue React

1. Instala Node.js.
2. Entra en la carpeta `frontend-react`.
3. Instala dependencias:
   ```bash
   npm install
   ```
4. Ejecuta la app:
   ```bash
   npm start
   ```
5. Accede a `http://localhost:3001` (o el puerto configurado)

Para producción:
- Ejecuta `npm run build` y sube la carpeta `build/` a un hosting estático (Vercel, Netlify, Firebase Hosting, etc).
